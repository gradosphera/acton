## `vm`

```tolk
struct vm
```

Holds the methods to change the VM state.

## `vm.getC7`

```tolk
fun vm.getC7(): tuple
    asm "c7 PUSHCTR";
```

Read control register C7.

## `vm.setC7`

```tolk
fun vm.setC7(c7: tuple): void
    asm "c7 POPCTR";
```

Write a new value to control register C7.

```tolk

```

